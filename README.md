# Clock

A shift scheduling and time tracking app inspired by Sling. Built with React + Vite.

## Features

- **Dashboard** - Overview of today's shifts, active clock-ins, weekly labor costs
- **Schedule** - Weekly calendar grid for managing employee shifts (create, edit, delete)
- **Employees** - Add, edit, and manage team members with roles and hourly rates
- **Time Clock** - Real-time clock in/out for employees with daily history
- **Settings** - Manage positions and app configuration

## Getting Started

```bash
npm install
npm run dev
```

## Tech Stack

- React 19
- React Router
- date-fns
- Lucide React (icons)
- Vite
